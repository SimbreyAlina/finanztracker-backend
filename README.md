# finanztracker-backend

## Finanztracker

Der Finanztracker ist eine Webanwendung zur übersichtlichen Verwaltung von persönlichen Ein- und Ausgaben.
Ziel der Anwendung ist es, einen schnellen Überblick über die eigene Finanzen zu bekommen. Nutzer können Transaktionen erfassen und verschiedenen Kategorien zuordnen – so lässt sich auf einen Blick erkennen, wofür Geld ausgegeben wird und woher die Einnahmen kommen.

### Funktionen
Mithilfe von Kategorien können Transaktionen gezielt eingeordnet und ausgewertet werden.


#### Ausgaben-Kategorien:

- Einkauf
- Essen gehen
- Spaß/Freizeit
- Geschenke
- Sonstiges


#### Einnahmen-Kategorien:

- Gehalt Hauptjob
- Gehalt Nebenjob
- Sonstiges


### Installation:

- Repository klonen: git clone git@github.com:SimbreyAlina/finanztracker-backend.git
- FinanztrackerApplication.java öffnen und starten
- Im Browser http://localhost:8080 aufrufen
